Flexbox Documentação

DISPLAY:FLEX;

A primeira propriedade que precisa ser declarada para que o Flexbox funcione! Define um container flex: inline ou blocos, ele é uni-dimensional, ou seja, podemos organizar horizontal ou verticalmente de forma a garatir que o comportem de maneira previsível oara evitar que saia do container de forma inesperada.

Algumas propriedades como flex-direction, justify-content e align-content já vem com os valores padrão

Propriedade Valor
flex-direction row
justify-content flex-start
align-content flex-start

Flex-wrap

"A propriedade flex-wrap define se os itens flexíveis são forçados a ficarem na mesma linha ou se podem ser quebradas em varias linhas. Se o argumento for valido, ele define a direção em que as linhas são empilhadas."

    Seus valore são:
  • Wrap - "Os itens flexíveis são quebrados em multiplas linhas";
  • Nowrap - "Os itens flexíveis são agrupados em uma unica linha, o que pode fazer com que o flex container transborde";
  • Wrap-reverse - "Se comporta da mesma maneira que o wrap, mas os itens flexíveis serão agrupados em várias linhas de baixo para cima.".
